⚡ DEPLOYMENT STEPS:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: MongoDB Atlas (5 min)
  → Sign up at MongoDB Atlas
  → Create FREE M0 cluster
  → Create user: blognest
  → Whitelist IP: 0.0.0.0/0
  → Get connection string
  → Add /blognest to the string

STEP 2: Deploy Backend (3 min)
  → Go to vercel.com/new
  → Import: NAYANSURYAVANSHI/blognest
  → Root Directory: backend
  → Add env vars:
     • MONGODB_URI = (your Atlas string)
     • JWT_SECRET = (generated secret)
     • NODE_ENV = production
  → Deploy
  → SAVE BACKEND URL

STEP 3: Deploy Frontend (3 min)
  → Go to vercel.com/new
  → Import: NAYANSURYAVANSHI/blognest
  → Root Directory: frontend
  → Add env var:
     • REACT_APP_API_URL = (backend URL)/api
  → Deploy
  → SAVE FRONTEND URL

STEP 4: Update Backend CORS (2 min)
  → Go to backend project settings
  → Add env var:
     • FRONTEND_URL = (frontend URL)
  → Redeploy backend

🆘 TROUBLESHOOTING:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

CORS Errors:
  → Check FRONTEND_URL is set in backend
  → Verify REACT_APP_API_URL in frontend
  → Redeploy both projects

Database Connection Failed:
  → Verify MongoDB connection string
  → Check IP whitelist (0.0.0.0/0)
  → Ensure password is correct

Build Failed:
  → Check build logs in Vercel
  → Verify all dependencies in package.json
  → Try redeploying
